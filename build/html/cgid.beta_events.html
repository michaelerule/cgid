cgid.beta_events module¶

cgid.beta_events.get_high_beta_events(session, area, channel, epoch, MINLEN=40, BOXLEN=50, THSCALE=1.5, lowf=10.0, highf=45.0, pad=200, clip=True, audit=False)[source]¶

get_high_beta_events(session,area,channel,epoch) will identify periods of elevated beta-frequency power for the given channel.

Thresholds are selected per-channel based on all available trials. The entire trial time is used when estimating the average beta power. To avoid recomputing, we extract beta events for all trials at once.

By default events that extend past the edge of the specified epoch will be clipped. Passing clip=False will discard these events.

returns the event threshold, and a list of event start and stop times relative to session time (not per-trial or epoch time)

passing audit=True will enable previewing each trial and the isolated beta events.

>>> thr,events = get_high_beta_events('SPK120925','PMd',50,(6,-1000,0))
